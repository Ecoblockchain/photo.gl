Proměny

float height = 150.0;
height = 160.0;

  • height je jméno proměny
  • 150.0 je hodnota
  • float je typ
  • řádek vždy končí ;

Proměna je krabice do které se dá uložit informace

Operace

float height = 150.0;
float shoeHeight = 10.0;
float totalHeight = height + shoeHeight;

  • height a shoeHeight jsou proměny
  • hodnota totalHeight je jejich součet

Pár operací: +, -, *, /

Podmínky

float shoeHeight = 10.0;
if ( height < 140.0 ) {
  shoeHeight = 20.0;
}

  • podmínka je jestlie: height < 140.0 - je to vlastně otázka
  • když je výsledek podmínky pravdivý, tak se exekutuje vše mezi { a }
  • jinak se { } přeskočí

Funkce

float totalHeight( height ) { // <- parametry
  float shoeHeight = 10.0;
  if ( height < 140.0 ) {
    shoeHeight = 20.0;
  }
  float result = height + shoeHeight;
  return result; // <- výsledek
}

Funkce

float felixHeight = totalHeight( 180.0 ); // = 190.0
float barboraHeight = totalHeight( 135.0 ) // = 155.0
